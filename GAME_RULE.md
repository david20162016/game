# 게임 규칙: Circle Survivor

> W, A, S, D로 동그라미를 조작하여 직선으로 날아오는 악당들을 피하세요!

## 🎮 게임 제목
**Circle Survivor**

---

## 📝 게임 설명
플레이어는 화면 중앙의 **파란색 동그라미**입니다.
사방에서 **빨간색 세모(악당)**들이 나타나 일직선으로 돌진합니다.
악당에 닿으면 게임 오버입니다. 최대한 오래 살아남으세요!

---

## 🎯 게임 목표
- **생존**: 가능한 오래 살아남아 높은 점수(생존 시간)를 기록하세요.

---

## 🕹️ 게임 규칙

### 1. 조작법
| 키보드 | 동작 |
|:---:|:---|
| **W** | 위로 이동 |
| **A** | 왼쪽으로 이동 |
| **S** | 아래로 이동 |
| **D** | 오른쪽으로 이동 |

### 2. 게임 요소
- **플레이어 (Circle)**:
    - 파란색 원.
    - 화면 밖으로 나갈 수 없음.
    - 초기 위치: 화면 중앙.
- **적 (Villain)**:
    - 빨간색 세모 (또는 원).
    - 화면 가장자리에서 스폰.
    - 반대편 끝을 향해 **일직선**으로 이동.
    - 속도는 시간이 지날수록 빨라짐.

### 3. 게임 오버 조건
- 플레이어가 적과 충돌하면 게임 종료.

---

## 💡 개발 참고 사항
1.  **Canvas 렌더링**:
    - HTML DOM 요소 대신 캔버스(`canvas`)에 직접 도형을 그려서 구현.
2.  **적 생성 로직**:
    - 일정 시간 간격(예: 1초)마다 화면 상하좌우 가장자리 중 랜덤 위치에서 생성.
    - 생성된 위치에서 화면 반대편 임의의 지점을 향해 벡터 계산 후 이동.
